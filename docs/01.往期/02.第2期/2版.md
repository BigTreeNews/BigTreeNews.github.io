## 来自地外文明的电报：接收地竟然是地球？
### 猎奇翻译院：新起的研究中心
5202年，一名天文爱好者在用精 密仪采观测星空时，偶然截获了一段似于有规律的电磁波，经过他缜密的计算，这道电磁波的发送位置在不断发生变化，但总体位置在银河系猎奇星臂的某一星系，他将其命名为9-12-Damn星系。第二日，他在同样的位置，同样的时间再次截获电磁波，他意识到世上可能真的有地外文明存在，于是他创立了猎奇翻译院，用以翻译电磁波内容，确保人类文明不受到来自地外文明的威胁。有意向加入者可拨打114-9191-4399。加入对未知的研究队伍！
### 院长安凌：会坚持从事翻译工作
“这是我的最大爱好。现在它变成了职业，我很开心，我会坚持从事翻译工作，争取翻译出更多有用信息，保障人类文明安全。”当记者采访安凌院长时，他如是说。安凌正是前文提到的天文爱好者，他为了攻克初步翻译工作，一度旁听了上海交大的物理电磁波学与清华大学的翻译学，结合他自身对天文学的热忱，使他出色地完成了初始语言翻译构造体系。“你们可能不相信这个体系所翻译出来的内容，但的确是这样。我相信有更多含义等待我们去发掘。”现在，安凌在进行翻译截获工作的同时也在旁听北大的星空观测学与天体探测学，希望这会对他的工作有所帮助。
### 星空下的电报究竟传达了什么？
（以下内容摘自安凌院长的工作报告）  
这是第一批电磁波翻译内容。发出地来自9-12-Damn星系，接收地是地球，由于发出地一直在星系内变换，导致内容时断时续，但可以肯定的是，是两个生物通过电磁波在进行交流！以下是翻译内容（接收者用P代名）
```
A：意大利面好用吗？
P：谢谢，效果不错，如果有熔岩酱就更好了。土豆们总里睡在镜片上，蓝牙不好吃。
A：早就跟你说过了，地像人很翔，回来吧。
P：你来，我给你做蛋仔派对
A：新词？听起来充满狗叫。
P：就这样吗？那人类……
A：不用，海龟会帮我们的。
P：乐，还有什么要吃的吗？
A：注意黄龙江，不要放沈阳大街。
P：小猴子怎么叫？
A：引擎在混凝土里，原理太难。
P：91。
A：明天晚安。
```
由此可见，外星人正在了解地球文明。我们不知其有什么阴谋，但防人之心不可无，所以将内容公之于众，欢迎各方人士积极讨论！
