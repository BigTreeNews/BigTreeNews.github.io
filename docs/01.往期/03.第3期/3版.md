## 餐盘占座被抓竟扣两分 如此治理乱象怎能根除？
这究竟是怎么一回事呢？带着这样的疑问，记者采访了当事人沈博，还原了当时的真相。  
11时46分，沈博如往常一样，平凡地来到食堂一楼买好饭菜。考虑到吃完饭再去杂货店会遇上午高峰，排队结账颇费时间，他随手把饭菜置于一餐桌上，迅速采购了饮料并返回。此时时间只有11时48分，沈博记得很清楚，他回来时一楼依旧有部分餐桌处子闲置状态。  
他坐下的一刹那，一神秘男子从侧门旁的阴暗角落窜出，指着他大骂：“你是不是占座？有没有公德心的啊……”当时沈博并未认出此人是谁，经后来的目击者提示他才知道该人正是叱咤杭七的学牲畜猪人——杨许枭（名号来自成语“自产自销”）
杨许枭气势正盛，沈博也不甘示弱。考虑到对方攻势猛烈，沈博也祭出了杀招——冷暴力，也就是他以此闻名的三不（不回应，不抵抗，不认错）。果不其然，杨许枭的势头被狠狠挫伤，它肉眼可见地红温了。双方经过激烈的交涉，从室内骂到户外，最后达成协议：沈博报上名号，以两分的代价换取交战时自己的餐盘占据餐桌13分钟，杨许枭将扣押的饮料归还沈博。  
沈博对这样的结果显然无法接受：“我只是将餐盘放在那里，其他人该有座也有座，为什么利已不损人的事情需要受到惩罚？”  
另有目击者反馈，杨许枭执法时的态度极其恶劣。他秉持疑罪从有的准则，凡是进来时拿着面包、饮料等物品的一律盘问，凡是用餐时桌上有饮料的一律批判。这样的行为充分展现了其作为学牲畜猪人的风范，生动表达出一条丧家之犬是怎么乱咬人的。  
记者认为，沈博出于临时放物品的目的而占座，做法虽略有欠缺，但实在不应该受到如此严厉的惩罚。真正该受到惩罚的是那些在食堂高峰期，连饭菜都没买好就用雨伞，水瓶甚至发卡等物品恶意挤占座位资源的人，这样的人难道不是更没公德心吗？占座乱象需要得到整治，但绝不能滥杀无辜，否则学牲畜的公信力只会趋近于零。  
罗总书记方面表示，即使沈博没有做了错范，他给班里造成的损失仍需要义务劳动（发配西伯利亚制造育碧服务器）两周以警示众人。
