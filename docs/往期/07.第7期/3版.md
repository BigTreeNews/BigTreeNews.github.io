## 甲醛中毒？不，是成神！娃哈哈高中四人中暑后，惊现“蓝电红超”双子侠！
近日，杭州娃哈哈双语学校新修了烹饪教室。同学们对其内部好奇万分，便推举Otto， Money，Sage，Linn四人组成“垃圾桶男子天团”，进入教室一探究竟。  
四人在一个夜深人静的中午光明正大地潜入烹饪教室，他们断断续续地鱼贯而入。Sage作为宝可梦全国赛亚军，是最后一个进入的。为了彰显娃哈哈纯净水的实力与狗尾巴草的礼貌，他随手将门关上了。谁知那门的设计师仿佛被皮卡丘的十万伏特电流击中，将门设计成了只能从外面打开，从里面出不去的啥博弈模式。四人就这样被困在了烹饪教室中！他们无心欣赏新建的设施与装备，因为屋内的甲醛明显超标。四人被熏得晕头转向，试图逃出。烹饪教室的窗户通向屋顶的阳台，“垃圾桶男子天团”成员只能瘦死的骆驼当活马医，破窗逃到了阳台上。  
但阳台上有着无数台空调外机，仿佛喷火的龙，要将四人焚烧。再加上烈日当空，四人纷纷中暑。无奈之下，四人只得大声呼救。一个小时后，一位勇者提着宝剑，打败了邪恶的烹饪教室，成功将四人救出。  
之后几天，四人完好无事。直到约一周后，Otto因高温出现了变异现象！只见他双眼发红，双手高举，手上还带着太阳的余温，微微发红，仿佛自己如太阳一般出现在中国的东方。而Money也未能幸免，他因吸入过量甲醛，双眼发蓝，向天上冲去，仿佛自己是蓝色妖姬，欲与将军比美。两人撞至一块，化为驴象，上演了一场红蓝争执。最终，两人摔倒在地，仿佛无事发生。  
而money仍怀恨在心。在之后的烹饪课上，老师教学生做“鱼香肉丝”。正在讲解时，money便以油为水，将半瓶辣椒油倒入锅中，做了一锅爆辣鱼香肉丝，又在下课之前溜出教室，将收拾的后事留给同学。  
记者有幸采访到了当事人Otto：
```
记者：能描述一下当时的场景吗？
Otto：我当时就听到咻～～～，崩的一声，真大声啦。
记者：谁把你救下来的？
Otto：压力马斯内。
记者：在烹饪教室中你的心情是什么样的？
Otto：开桂了，你真能拒绝和我再去一次吗？
记者：我想还是算了，兄弟。
Otto：你都叫我兄弟了那我说啥了。
```
由此可见，Otto的恢复并不完全。让我们为他默默祈祷，希望娃哈哈的纯净水能让他们重回地球人。
